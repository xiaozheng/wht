== Description ==

WordPress Post From Frontend Plugin is to allow user to create and manage posts, edit profiles info from front-end.

= Usage = 
what all you have to do is add short-code [wpeden_post_from_frontend] in a page or post content after install and activate the plugin


== Installation ==


1. Upload `post-from-frontend` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
